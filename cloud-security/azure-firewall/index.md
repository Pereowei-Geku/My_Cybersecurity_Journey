# Azure Firewall Implementation

Deployed Azure Firewall as a centralized network security solution to control network traffic for Prafydob Corp.

## What I Built

- Virtual network with 3 isolated subnets
- Azure Firewall for traffic filtering
- Custom routing to force traffic through firewall
- Security rules for web access and DNS

## Key Results

✅ Successfully blocked unauthorized websites  
✅ Only allowed www.bing.com access  
✅ Implemented defense-in-depth security  
✅ Validated zero-trust architecture  

## Technologies Used

- Azure Firewall
- Virtual Networks
- User-Defined Routes
- Windows Server 2025
